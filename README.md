# Sentinel

A Discord utility bot (TypeScript, pnpm).

## Requirements

- Node.js `>=22 <26`
- pnpm `>=10.28.2`

## Setup

1. `pnpm install`
2. Copy `.env.example` to `.env` and fill in the values.
3. `pnpm build` to compile TypeScript do `dist/`.
4. `pnpm start` to run compiled `dist/` (loads `.env` if present).
